# 🚀 AI Job Copilot

AI-powered job search assistant that:

- Parses resumes using LLMs
- Stores embeddings in Pinecone XXX
- Retrieves profile via RAG
- Scrapes real job listings
- Matches jobs to candidate profile

## Tech Stack

- FastAPI
- Groq LLM
- Pinecone Vector DB
- Pyppeteer Scraper
- Streamlit Frontend
